# 关于 ClawTools

## 什么是 ClawTools？

ClawTools 是一款面向 [OpenClaw](https://github.com/openclaw/openclaw) 的开源管理套件，旨在简化 OpenClaw 在不同平台上的安装、配置和维护工作。

## 目标

- **简化安装** - 一条命令在任意平台安装 OpenClaw
- **集中管理** - 在一个地方管理所有 OpenClaw 配置
- **跨平台** - 在 Windows、Linux、macOS 等平台上无缝工作
- **面向开发者** - 由开发者打造，为开发者服务

## 为什么选择 ClawTools？

OpenClaw 是一款运行在本地设备上的强大 AI 助手。然而，在不同平台上管理它、配置提供商、维护备份等工作可能很复杂。ClawTools 通过以下方式解决这些挑战：

1. **统一界面** - 一个友好的 TUI 完成所有 OpenClaw 管理任务
2. **安全操作** - 内置诊断和备份功能
3. **多提供商支持** - 轻松配置各种 API 提供商
4. **离线文档** - 无需网络即可搜索 OpenClaw 文档

## 设计理念

- **简单至上** - 复杂任务应该简单化
- **透明可控** - 所有操作可见可管
- **安全优先** - 通过备份和安全默认值保护用户数据
- **模块化** - 易于扩展和定制

## 技术栈

- **Node.js** - 运行时环境
- **TypeScript** - 类型安全开发
- **Ink** - 基于 React 的 TUI 框架
- **SQLite** - 本地数据存储
- **FlexSearch** - 全文搜索
