# About ClawTools

## What is ClawTools?

ClawTools is an open-source management suite for [OpenClaw](https://github.com/openclaw/openclaw), designed to simplify the installation, configuration, and maintenance of OpenClaw across different platforms.

## Goals

- **Simplify Installation** - One-command installation for OpenClaw on any platform
- **Centralized Management** - Manage all OpenClaw configurations in one place
- **Cross-Platform** - Work seamlessly on Windows, Linux, macOS, and more
- **Developer-Friendly** - Built with developers in mind, for developers

## Why ClawTools?

OpenClaw is a powerful AI assistant that runs on your device. However, managing it across different platforms, configuring providers, and maintaining backups can be complex. ClawTools addresses these challenges by providing:

1. **Unified Interface** - A friendly TUI for all OpenClaw management tasks
2. **Safe Operations** - Built-in diagnostics and backup capabilities
3. **Multi-Provider Support** - Easy configuration for various API providers
4. **Offline Documentation** - Search OpenClaw docs without internet

## Design Philosophy

- **Simplicity First** - Complex tasks should be simple to perform
- **Transparency** - All operations are visible and controllable
- **Safety** - Protect user data with backups and safe defaults
- **Modularity** - Easy to extend and customize

## Technology Stack

- **Node.js** - Runtime environment
- **TypeScript** - Type-safe development
- **Ink** - React-based TUI framework
- **SQLite** - Local data storage
- **FlexSearch** - Full-text search
